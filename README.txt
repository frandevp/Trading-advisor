# AI Trading Advisor

AI trading advisor built with Python to analyze historical stock data and generate insights using Ollama.

## Features

* Analyze historical stock data
* Data processing with pandas
* AI-generated insights using Ollama
* Simple trading advisor prototype

## Tech Stack

* Python
* pandas
* Ollama

## Project Structure

.
├── traderbot/              # Trading bot logic and modules
├── advisor_local_nvda.py   # AI advisor script for NVDA stock analysis
└── README.md               # Project documentation

## Setup

Create a virtual environment and install dependencies:

pip install pandas

Make sure Ollama is installed and running locally.

## Usage

Run the advisor script:

python advisor_local_nvda.py
